# The Ardhi — Property Management Platform

Built for Uganda. Three-tier: Admin, Landlord, Tenant.
Warm, approachable design. Mobile-first. Professional icon pack.

## Live
Deployed on Vercel. Connected to Supabase.

## Stack
- Next.js 16, TypeScript
- Supabase (Auth + PostgreSQL + RLS)
- Lucide React icons
- Deployed on Vercel

## Portals
- `/` — Landing + Auth (Admin, Landlord, Tenant)
- `/landlord` — Property management, tenant onboarding, payments
- `/tenant` — Pay rent, complaints, community alerts
- `/admin` — Full platform oversight
